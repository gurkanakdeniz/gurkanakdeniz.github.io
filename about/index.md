<center><a href="http://gurkanakdeniz.github.io/"><b>Rocket Space</b></a> is a jekyll theme.</center>

## Features
* Responsive
* Disqus integration
* Syntax highlighting
* Optional post image
* Social icons
* Page for sharing projects
* Optional background image
* Simple navigation menu
* MathJax support
